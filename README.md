Этот репозиторий — набор pet-проектов, в которых я демонстрирую навыки анализа данных и машинного обучения. 

Каждый проект — это попытка решить практическую задачу, протестировать подходы и лучше понять, как работают инструменты на реальных или приближенных к реальности данных.

Все проекты имеют отдельные папки и, как правило, включают:

- описание задачи

- исследовательский анализ данных

- подготовку данных

- моделирование

- выводы
