Прогнозирование качества вина с использованием алгоритма KNN

# Описание проекта

Целью данного проекта является анализ химических характеристик вина и построение модели машинного обучения для прогнозирования его качества

В проекте используется датасет Wine Quality (https://www.kaggle.com/datasets/amirhosseinzandi/red-and-white-wine-quality?utm_source=chatgpt.com), который физико-химические параметры красных и белых вин. На основе этих параметров строится модель классификации, определяющая категорию качества вина.

Качество вина в исходном датасете представлено числовой шкалой. Для упрощения задачи классификации оценки были сгруппированы в три категории:

Bad — низкое качество (оценка ≤ 5)

Average — среднее качество (оценка = 6)

Good — высокое качество (оценка ≥ 7)

Такое преобразование позволяет сделать задачу более интерпретируемой и удобной для применения алгоритмов машинного обучения.

# EDA

Перед обучением модели был проведён анализ структуры данных и взаимосвязей между признаками. Анализ распределения показал, что превалирующее число вин в датасете относятся к категории среднего качества. Количество вин высокого качества значительно меньше, что усложняет задачу классификации.

Корреляционная матрица позволяет определить взаимосвязи между физико-химическими параметрами вина.

Наиболее заметные зависимости:

положительная корреляция между residual sugar и density

отрицательная корреляция между alcohol и density

Связь признаков с качеством вина

Наиболее сильная положительная корреляция с качеством наблюдается у признака alcohol

Наиболее заметная отрицательная связь наблюдается у volatile acidity и density

Это означает, что увеличение содержания алкоголя часто связано с более высоким качеством вина, тогда как высокая летучая кислотность может ухудшать оценку.

# Построение модели

Для решения задачи классификации был выбран алгоритм K-ближайших соседей (KNN).

Алгоритм определяет класс объекта на основе классов его ближайших соседей в пространстве признаков.

Этапы построения модели:

1. Подготовка данных

2. Масштабирование признаков

3. Разделение данных на обучающую и тестовую выборки

4. Подбор оптимального значения параметра k

5. Обучение модели

# Выбор K

Для выбора оптимального параметра была проведена серия экспериментов. Был сформирован диапазон значений k (от 1 до 20). Для каждого значения обучалась модель, а после рассчитывалась точность на тестовой выборке. Результаты сохранялись и строился график зависимости точности от k.
График показывает, как меняется точность модели при изменении количества соседей.

По результатам эксперимента наилучшую точность показало значение k = 1. Приходим к выводу, что ближайший сосед содержит достаточно информации для определения категории качества вина. Также такой результат может объясняться тем, что химические характеристики вина образуют достаточно плотные кластеры в пространстве признаков.

# Оценка модели

Матрица ошибок показывает распределение правильных и неправильных классификаций. Основные ошибки возникают между категориями average и good, что объясняется схожими химическими характеристиками этих вин.

Итоговая точность модели составила около 0.66, что является типичным результатом для алгоритма KNN на данном датасете.
