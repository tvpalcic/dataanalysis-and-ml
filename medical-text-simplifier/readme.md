Medical Text Simplifier — это десктопное приложение на Python с базовым графическим интерфейсом, которое упрощает сложные медицинские тексты для людей без медицинского образования. Приложение использует дообученную кастомную модель GPT-3.5, чтобы сохранять смысл текста и сделать его максимально понятным для непрофессионалов.

Основная идея
Мини приложение в которое можно загрузить сложный медицинский отчет или заключение и получить упрощенную, но не искаженную версию для непрофессионала.

Обучение модели 
Был взят датасет данных в формате Оригинальный текст - Упрощенный текст (человек) - Упрощенный текст (ИИ). Fine-tuning модель OpenAI 3.5 turbo обучалась облегчать текст на этих данных без потери смысла. Были три выборки с разными гиперпараметрами (epochs, batch size, LR multiplier). Для честности эксперимента seed был везде одинаковый. Выбран наиболее оптимальный вариант.

Реализация
Эту модель мы подключали через API к базовому интерфейсу сделаному при помощи библотеки PyQt5. Еще парочкой библиотек мы реализовываем загрузку файлов разных форматов и получаем выходной результат. Проект собирался благодаря PyInstaller в .exe приложение. 

Функционал
Графический инерфейс делит окно приложения на три зоны: 
- слева окно истории запросов, которые хранятся в отдельном файле. Реализована кнопка очистки истории
- по центру основное окно запроса, в которое можно ввести данные вручную, а можно загрузить посредством отдельных кнопок txt/pdf/docx файл, по всем кнопка упрощения, которая отправляет данные на сервер
- справа окно результатов, с возможностью сохранить полученный ответ в txt файл. 

Все легко пересобрать с другими данными, например на других языках, при наличии походящего датасета
